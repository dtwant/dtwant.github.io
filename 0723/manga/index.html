JSONBIN SYNC
jsonbin.io を使ってデータを安全にクラウド保存 (エラー回避済)できます。初回は自動でBlobが作成されます。
—

仕様および運用手順
#

基本操作

  • ＋ 追加 : タイトル、作者、ステータス、評価、タグ、メモなどの各種メタデータを入力し、「SAVE」にて登録を実行する。
  • カードの選択 : 当該漫画の詳細情報を表示する。詳細画面より「編集（✎ 編集）」および「削除（🗑 削除）」の操作が可能である。
  • フィルタ・ソート機能 : 画面上部のタブ（読了・読中など）や検索窓よりデータを絞り込む。また、更新順や評価順での並び替えに対応する。

クラウド同期 (jsonbin.io)

  1. 初期状態において、自動的に新規の Bin (ストレージ領域) が作成される。
  2. 画面右上部の ⚙ CONFIG を選択し、表示される Blob ID (Bin ID) を必ず控えること。
  3. 他端末へデータを引き継ぐ際は、CONFIG 画面の入力欄に該当の Blob ID を入力のうえ「APPLY」を実行する。
  4. 設定後は ⇅ SYNC を押下し、最新のデータを同期する。

データは jsonbin.io 上に保存される。 認証キーと連動した単一のIDで管理されるため、Blob ID を紛失した場合、既存データへのアクセスは不可能となる。厳重に管理すること。

ローカルバックアップ

CONFIG 画面より JSON EXPORT を実行することで、現在のデータをローカルデバイスへ保存可能である。 復元の際は JSON IMPORT より該当の JSON ファイルを読み込み、再度 SYNC を実行すること。
