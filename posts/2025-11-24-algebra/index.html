序
#

表記において, 集合 $X$ の濃度を $|X|$ または $\sharp X$ と記す. 特記がない限り, 環は単位元を持つ可換環とする.

第1部 群論 (Group Theory)
#

1. 定義
#

1.1 群の公理
#

集合 $G$ および二項演算 $\cdot: G \times G \to G$ の組 $(G, \cdot)$ が以下の公理を満たすとき, これを群という.

  1.  結合律: 任意の $a, b, c \in G$ に対し $(ab)c = a(bc)$ が成立する.
  2.  単位元の存在: ある元 $e \in G$ が存在し, 任意の $a \in G$ に対し $ae = ea = a$ を満たす.
  3.  逆元の存在: 任意の $a \in G$ に対し, ある元 $b \in G$ が存在して $ab = ba = e$ を満たす. $b$ を $a^{-1}$ と記す.

さらに, 任意の $a, b \in G$ に対し $ab = ba$ が成立するとき, $G$ をアーベル群 (可換群) という.

1.2 部分群と生成系
#

$G$ の部分集合 $H$ が $G$ の演算に関して群を成すとき, $H \le G$ と記す. $S \subset G$ に対し, $S$ を含む最小の部分群を $\langle S \rangle$ と表記し, $S$ で生成される部分群という. $G = \langle S \rangle$ となる有限集合 $S$ が存在するとき, $G$ を有限生成群という.

巡回群: $G = \langle g \rangle$ を満たす元 $g \in G$ が存在する群. $\mathbb{Z}$ または $\mathbb{Z}/n\mathbb{Z}$ に同型である.

2. 剰余類分解と正規部分群
#

2.1 Lagrangeの定理
#

$H \le G$ とする. $G$ 上の関係 $a \sim b \iff a^{-1}b \in H$ は同値関係であり, 同値類 $aH = {ah \mid h \in H}$ を左剰余類という. 左剰余類の集合を $G/H$ と書き, その濃度を指数 $[G:H]$ と呼ぶ.

定理 (Lagrange) $G$ が有限群ならば, $|G| = [G:H] \cdot |H|$ が成立する. 特筆すべき系として, 部分群の位数は群の位数の約数である.

2.2 正規部分群と剰余群
#

部分群 $N \le G$ が 正規部分群 ($N \trianglelefteq G$) であるとは, 任意の $g \in G, n \in N$ に対し $gng^{-1} \in N$ が成立することである. このとき, 剰余類の集合 $G/N$ に演算 $(aN)(bN) = (ab)N$ が well-defined となり, 群構造が入る. これを剰余群という.

3. 準同型定理と同型定理
#

群準同型 $\phi: G \to G’$ に対し, $\mathrm{Ker}(\phi) = {g \in G \mid \phi(g) = e’}$ は $G$ の正規部分群である.

定理 (準同型定理)

$$G/\mathrm{Ker}(\phi) \cong \mathrm{Im}(\phi)$$

定理 (同型定理)

  1.  $H \le G, N \trianglelefteq G$ ならば $HN/N \cong H/(H \cap N)$.
  2.  $N, K \trianglelefteq G$ かつ $N \subset K$ ならば $(G/N)/(K/N) \cong G/K$.

4. 群の作用とSylowの定理
#

4.1 群作用と軌道・固定部分群
#

群 $G$ の集合 $X$ への作用とは, 写像 $G \times X \to X, (g, x) \mapsto gx$ であり, 結合律と単位元公理を満たすものである.

  • 軌道 (Orbit): $O_x = {gx \mid g \in G}$
  • 固定部分群 (Stabilizer): $G_x = {g \in G \mid gx = x}$

軌道・固定群定理: $|O_x| = [G : G_x]$

4.2 Sylowの定理
#

有限群 $G$ の位数を $|G| = p^k m$ ($p$ は素数, $p \nmid m$) とする.

  1.  存在定理: 位数 $p^k$ の部分群 (Sylow $p$-部分群) が存在する.
  2.  共役定理: 任意の Sylow $p$-部分群は互いに共役である.
  3.  個数定理: Sylow $p$-部分群の個数 $n_p$ は, $n_p \equiv 1 \pmod p$ かつ $n_p \mid m$ を満たす.

5. 有限アーベル群の構造定理
#

定理 (有限生成アーベル群の基本定理) 有限生成アーベル群 $G$ は, 以下の通り巡回群の直積に一意的に分解される.

$$G \cong \mathbb{Z}^r \oplus \mathbb{Z}/d_1\mathbb{Z} \oplus \dots \oplus \mathbb{Z}/d_k\mathbb{Z}$$

ここで $r \ge 0$ はランク, $d_1 \mid d_2 \mid \dots \mid d_k$ は単因子と呼ばれる.

第2部 環論 (Ring Theory)
#

1. 環とイデアル
#

1.1 定義
#

環 $R$ の部分集合 $I$ が イデアル であるとは, $I$ が加法群であり, かつ $\forall r \in R, \forall x \in I \Rightarrow rx \in I$ を満たすことをいう. イデアルにより剰余環 $R/I$ が定義される.

1.2 素イデアルと極大イデアル
#

  • 素イデアル (Prime Ideal) $P$: $xy \in P \Rightarrow x \in P \lor y \in P$. これは $R/P$ が整域であることと同値.
  • 極大イデアル (Maximal Ideal) $M$: $M \subsetneq R$ であり, $M$ を含む真のイデアルが存在しない. これは $R/M$ が体であることと同値.

Zornの補題より, 単位元を持つ環には少なくとも一つの極大イデアルが存在する (Krullの定理).

2. 整域のクラス
#

整域 (零因子を持たない可換環) の包含関係は以下の通りである.

$$\text{体} \subsetneq \text{ユークリッド整域 (ED)} \subsetneq \text{単項イデアル整域 (PID)} \subsetneq \text{一意分解整域 (UFD)} \subsetneq \text{整域}$$

2.1 主な性質
#

  • UFD: 素元分解の一意性が成り立つ (例: $\mathbb{Z}, K[x_1, \dots, x_n]$).
  • PID: 全てのイデアルが単項 $I=(a)$ で生成される (例: $\mathbb{Z}, K[x]$).
  • ED: 除法の原理が成り立つ (例: $\mathbb{Z}, \mathbb{Z}[i]$).

定理: PID上では, 素イデアル $\neq 0$ は極大イデアルである.

3. 多項式環とNoether環
#

3.1 Noether環
#

以下の同値な条件を満たす環 $R$ を Noether 環という.

  1.  昇鎖条件: イデアルの昇鎖 $I_1 \subset I_2 \subset \dots$ は有限回で停止する.
  2.  任意のイデアルは有限生成である.

3.2 Hilbertの基底定理
#

定理: $R$ が Noether 環ならば, 一変数多項式環 $R[x]$ も Noether 環である. 帰納的に $R[x_1, \dots, x_n]$ も Noether 環となる.

4. 局所化 (Localization)
#

環 $R$ と積閉集合 $S$ ($1 \in S, 0 \notin S, \forall s,t \in S \Rightarrow st \in S$) に対し, 分数環 $S^{-1}R = { r/s \mid r \in R, s \in S }$ を構成する. 特に素イデアル $P$ に対し $S = R \setminus P$ とした $R_P$ は, 唯一の極大イデアル $PR_P$ を持つ局所環となる.

第3部 体論とガロア理論 (Field Theory & Galois Theory)
#

1. 体の拡大
#

1.1 代数拡大と超越拡大
#

体 $K$ の拡大体 $L$ の元 $\alpha$ が $K$ 上代数的であるとは, $f(\alpha)=0$ となる $f(x) \in K[x]$ が存在することを指す. 存在しない場合を超越的という. $L$ の全ての元が $K$ 上代数的であるとき, $L/K$ を代数拡大という.

1.2 最小多項式と拡大次数
#

代数的元 $\alpha$ に対し, $f(\alpha)=0$ を満たすモニック既約多項式を最小多項式という. 拡大次数 $[L:K] = \dim_K L$ について, 積の法則 $[M:K] = [M:L][L:K]$ が成立する.

2. 分解体と正規性・分離性
#

2.1 分解体
#

多項式 $f(x) \in K[x]$ が $L$ 上で一次式の積に分解され, かつその根で生成されるとき, $L$ を $f$ の分解体という.

2.2 正規拡大 (Normal Extension)
#

既約多項式が一つ根を持てば, 全ての根を持つような拡大. あるいは, 有限次拡大において分解体であることと同値.

2.3 分離拡大 (Separable Extension)
#

各元の最小多項式が重根を持たない拡大. 標数0の体や有限体上の代数拡大は常に分離的である.

3. ガロア理論の基本定理
#

定義 (ガロア拡大) $L/K$ が正規かつ分離的であるとき, これをガロア拡大と呼ぶ. このとき, 自己同型群 $\mathrm{Aut}(L/K)$ をガロア群 $\mathrm{Gal}(L/K)$ と定義する.

定理 (ガロア対応) $L/K$ を有限次ガロア拡大とする. 中間体 $M$ ($K \subset M \subset L$) とガロア群の部分群 $H \le \mathrm{Gal}(L/K)$ の間には以下の一対一対応が存在する.

$$M \longleftrightarrow \mathrm{Gal}(L/M)$$

$$L^H \longleftrightarrow H$$
  1.  $[L:M] = |\mathrm{Gal}(L/M)|$
  2.  $[M:K] = [G : \mathrm{Gal}(L/M)]$
  3.  $M/K$ が正規拡大 $\iff \mathrm{Gal}(L/M) \trianglelefteq \mathrm{Gal}(L/K)$. このとき $\mathrm{Gal}(M/K) \cong \mathrm{Gal}(L/K) / \mathrm{Gal}(L/M)$.

4. 方程式の可解性
#

定理 (Abel-Ruffini) 代数方程式 $f(x)=0$ が代数的 (四則演算と冪根) に可解である $\iff$ ガロア群 $G_f$ が可解群である.

定義 (可解群) 群 $G$ が可解群であるとは, 正規部分群の列 $1 = G_0 \trianglelefteq G_1 \trianglelefteq \dots \trianglelefteq G_k = G$ が存在し, 各剰余群 $G_{i+1}/G_i$ がアーベル群となることである.

$n \ge 5$ のとき, 対称群 $S_n$ は可解群ではない (交代群 $A_n$ が単純群であるため). したがって, 5次以上の一般方程式には代数的な解の公式が存在しない.

5. 有限体 (Finite Fields)
#

有限体 $\mathbb{F}_q$ は位数が素数冪 $q=p^n$ の場合に限り存在し, 同型を除いて一意である.

  1.  $\mathbb{F}_{p^n}$ は $\mathbb{F}_p$ 上の多項式 $x^{p^n} - x$ の分解体である.
  2.  乗法群 $\mathbb{F}_{p^n}^\times$ は位数 $p^n-1$ の巡回群である.
  3.  ガロア群 $\mathrm{Gal}(\mathbb{F}_{p^n}/\mathbb{F}_p)$ は, フロベニウス写像 $x \mapsto x^p$ で生成される位数 $n$ の巡回群である.

付録: 加群 (Modules)
#

A.1 定義
#

環 $R$ 上の加群 $M$ とは, ベクトル空間の定義における「体」を「環」に置き換えた代数系である. $R$ が体でない場合, 基底が存在するとは限らない (自由加群とは限らない).

A.2 完全列と蛇の補題
#

加群の準同型 $A \xrightarrow{f} B \xrightarrow{g} C$ において $\mathrm{Im}(f) = \mathrm{Ker}(g)$ が成立するとき, これを完全列という. 短完全列 $0 \to A \to B \to C \to 0$ はホモロジー代数の基礎を成す.

蛇の補題 (Snake Lemma): 可換図式における核と余核の間に完全列を誘導する補題. 5項補題等とともに, 図式追跡 (Diagram Chasing) の基本的な道具である.
