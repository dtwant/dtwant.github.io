ラプラス変換の概要
#

ラプラス変換は、ある関数 $f(t)$ を別の関数 $F(s)$ に変換する手法の一つです 。特に、微分方程式を代数方程式に変換できるため、工学や物理学の分野で広く利用されます。

1. 定義
#

$t \ge 0$ で定義された関数 $f(t)$ のラプラス変換 $\mathcal{L}[f(t)]$ は、以下の積分で定義されます 。このとき、$s$ は複素数ですが、ここでは実数として扱います。

$$F(s) = \mathcal{L}[f(t)] = \int_0^\infty e^{-st} f(t) dt \text{ }$$

この積分が収束するような $s$ の範囲において、$F(s)$ は定義されます 。

2. ラプラス変換の線形性
#

ラプラス変換は線形性を持っています。つまり、2つの関数 $f(t)$ と $g(t)$ および定数 $k$ に対して、以下の性質が成り立ちます 。

  • 和の法則: $\mathcal{L}[f(t) + g(t)] = \mathcal{L}[f(t)] + \mathcal{L}[g(t)]$
  • 定数倍: $\mathcal{L}[k f(t)] = k \mathcal{L}[f(t)]$

（計算過程） 線形性は積分の性質から直接導かれます 。

$$ \begin{aligned} \mathcal{L}[f(t) + g(t)] &= \int_0^\infty e^{-st} \{f(t) + g(t)\} dt \\ &= \int_0^\infty e^{-st}f(t)dt + \int_0^\infty e^{-st}g(t)dt \\ &= \mathcal{L}[f(t)] + \mathcal{L}[g(t)] \text{ } \end{aligned} $$

$$ \begin{aligned} \mathcal{L}[k f(t)] &= \int_0^\infty e^{-st} \{k f(t)\} dt \\ &= k \int_0^\infty e^{-st}f(t)dt \\ &= k \mathcal{L}[f(t)] \text{ } \end{aligned} $$

基本的な関数のラプラス変換
#

いくつかの基本的な関数のラプラス変換は、定義式から直接計算できます。

  • 定数関数 $f(t) = 1$     $$     \mathcal{L}[1] = \frac{1}{s} \quad (s>0) \text{ }     $$

    （計算過程）    

$$     \mathcal{L}[1] = \int_0^\infty e^{-st} \cdot 1 dt = \left[ -\frac{1}{s}e^{-st} \right]_0^\infty = \lim_{T \to \infty} \left( -\frac{1}{s}e^{-sT} \right) - \left( -\frac{1}{s}e^{0} \right) = 0 - \left(-\frac{1}{s}\right) = \frac{1}{s} \quad (s>0) \text{ }     $$
  • べき関数 $f(t) = t$     $$     \mathcal{L}[t] = \frac{1}{s^2} \quad (s>0) \text{ }     $$

    （計算過程）     部分積分を用いて計算します 。    

$$     \begin{aligned}     \mathcal{L}[t] &= \int_0^\infty t e^{-st} dt \\     &= \left[ t \left( -\frac{1}{s}e^{-st} \right) \right]_0^\infty - \int_0^\infty 1 \cdot \left( -\frac{1}{s}e^{-st} \right) dt \text{ } \\     &= \lim_{T \to \infty} \left\{ \left[-\frac{t}{s}e^{-st}\right]_0^T \right\} + \frac{1}{s} \int_0^\infty e^{-st} dt \text{ } \\     &= 0 + \frac{1}{s} \mathcal{L}[1] = \frac{1}{s^2} \quad (s>0) \text{ }     \end{aligned}     $$
  • 自然数 $n$ に対する $f(t) = t^n$     $$     \mathcal{L}[t^n] = \frac{n!}{s^{n+1}} \quad (s>0) \text{ }     $$

    （計算過程）     数学的帰納法と部分積分により証明されます 。$m=k$で成立を仮定し、$m=k+1$の場合を計算します。    

$$     \begin{aligned}     \mathcal{L}[t^{k+1}] &= \int_0^\infty t^{k+1}e^{-st}dt \\     &= \left[ t^{k+1} \left(-\frac{1}{s}e^{-st}\right) \right]_0^\infty - \int_0^\infty (k+1)t^k \left(-\frac{1}{s}e^{-st}\right) dt \text{ } \\     &= 0 + \frac{k+1}{s} \int_0^\infty t^k e^{-st} dt \\     &= \frac{k+1}{s} \mathcal{L}[t^k] = \frac{k+1}{s} \frac{k!}{s^{k+1}} = \frac{(k+1)!}{s^{k+2}} \text{ }     \end{aligned}     $$

   

  • 指数関数 $f(t) = e^{at}$     $$     \mathcal{L}[e^{at}] = \frac{1}{s-a} \quad (s>a) \text{ }     $$

    （計算過程）    

$$     \mathcal{L}[e^{at}] = \int_0^\infty e^{-st} e^{at} dt = \int_0^\infty e^{-(s-a)t} dt = \left[ \frac{1}{-(s-a)}e^{-(s-a)t} \right]_0^\infty = \frac{1}{s-a} \quad (s>a) \text{ }     $$

   

  • 三角関数     $$     \mathcal{L}[\sin(\omega t)] = \frac{\omega}{s^2 + \omega^2} \quad (s>0) \text{ }     $$     $$     \mathcal{L}[\cos(\omega t)] = \frac{s}{s^2 + \omega^2} \quad (s>0) \text{ }     $$

    （計算過程： $\sin(\omega t)$）     部分積分を2回利用します 。    

$$     \begin{aligned}     \mathcal{L}[\sin(\omega t)] &= \int_0^\infty e^{-st}\sin(\omega t) dt \\     &= \left[ \sin(\omega t) \left(-\frac{1}{s}e^{-st}\right) \right]_0^\infty - \int_0^\infty \omega\cos(\omega t) \left(-\frac{1}{s}e^{-st}\right) dt \\     &= \frac{\omega}{s} \int_0^\infty e^{-st}\cos(\omega t) dt \text{ } \\     &= \frac{\omega}{s} \left( \left[ \cos(\omega t) \left(-\frac{1}{s}e^{-st}\right) \right]_0^\infty - \int_0^\infty (-\omega\sin(\omega t)) \left(-\frac{1}{s}e^{-st}\right) dt \right) \\     &= \frac{\omega}{s} \left( \frac{1}{s} - \frac{\omega}{s} \int_0^\infty e^{-st}\sin(\omega t) dt \right) \text{ } \\     &= \frac{\omega}{s^2} - \frac{\omega^2}{s^2}\mathcal{L}[\sin(\omega t)] \text{ }     \end{aligned}     $$

    これを $\mathcal{L}[\sin(\omega t)]$ について解くと、$(1+\frac{\omega^2}{s^2})\mathcal{L}[\sin(\omega t)] = \frac{\omega}{s^2}$ となり、$\mathcal{L}[\sin(\omega t)] = \frac{\omega}{s^2+\omega^2}$ が得られます 。

    （計算過程： $\cos(\omega t)$ via Euler）     オイラーの公式 $e^{ibt} = \cos(bt) + i\sin(bt)$ を利用すると、$\mathcal{L}[\cos(bt)]$ は $\mathcal{L}[e^{ibt}]$ の実部となります 。    

$$     \begin{aligned}     \mathcal{L}[e^{ibt}] &= \frac{1}{s-ib} = \frac{s+ib}{(s-ib)(s+ib)} = \frac{s+ib}{s^2+b^2} \\     &= \frac{s}{s^2+b^2} + i \frac{b}{s^2+b^2} \text{ }     \end{aligned}     $$

    実部と虚部を比較することで、$\mathcal{L}[\cos(bt)] = \frac{s}{s^2+b^2}$ と $\mathcal{L}[\sin(bt)] = \frac{b}{s^2+b^2}$ が同時に得られます 。    

  • 双曲線関数     $$     \mathcal{L}[\cosh(bt)] = \frac{s}{s^2 - b^2} \quad (s>|b|) \text{ }     $$     $$     \mathcal{L}[\sinh(bt)] = \frac{b}{s^2 - b^2} \quad (s>|b|) \text{ }     $$

    （計算過程）     定義式と線形性を利用します 。    

$$     \begin{aligned}     \mathcal{L}[\cosh(bt)] &= \mathcal{L}\left[\frac{e^{bt}+e^{-bt}}{2}\right] = \frac{1}{2}(\mathcal{L}[e^{bt}] + \mathcal{L}[e^{-bt}]) \\     &= \frac{1}{2}\left(\frac{1}{s-b} + \frac{1}{s+b}\right) = \frac{1}{2}\frac{(s+b)+(s-b)}{s^2-b^2} = \frac{s}{s^2-b^2} \text{ }     \end{aligned}     $$

    $\sinh(bt)$ も同様に計算できます 。

ラプラス変換の重要な性質
#

1. 導関数のラプラス変換
#

  • 一階導関数:     

    $$     \mathcal{L}[f'(t)] = sF(s) - f(0) \text{ }     $$

        （計算過程）     部分積分により証明します 。    

    $$     \begin{aligned}     \mathcal{L}[f'(t)] &= \int_0^\infty e^{-st}f'(t)dt \\     &= \left[ e^{-st}f(t) \right]_0^\infty - \int_0^\infty (-se^{-st})f(t)dt \text{ } \\     &= (0 - e^0 f(0)) + s \int_0^\infty e^{-st}f(t)dt \text{ } \\     &= sF(s) - f(0) \text{ }     \end{aligned}     $$

  • n階導関数:     

    $$     \mathcal{L}[f^{(n)}(t)] = s^n F(s) - s^{n-1}f(0) - s^{n-2}f'(0) - \cdots - f^{(n-1)}(0) \text{ }     $$

2. 積分のラプラス変換
#

$$\mathcal{L}\left[\int_0^t f(u)du\right] = \frac{F(s)}{s} \text{ }$$

（計算過程） $g(t) = \int_0^t f(u)du$ とおくと、$g’(t)=f(t), g(0)=0$ となります 。導関数の変換公式を適用すると、

$$ \mathcal{L}[f(t)] = \mathcal{L}[g'(t)] = s\mathcal{L}[g(t)] - g(0) = s\mathcal{L}\left[\int_0^t f(u)du\right] \text{ } $$

よって、$F(s) = s\mathcal{L}\left[\int_0^t f(u)du\right]$ となり、式が導かれます 。

3. 移動定理
#

  • 第一移動定理 (s-shifting):     

    $$     \mathcal{L}[e^{at}f(t)] = F(s-a) \text{ }     $$

        （計算過程）    

    $$     \mathcal{L}[e^{at}f(t)] = \int_0^\infty e^{-st}e^{at}f(t)dt = \int_0^\infty e^{-(s-a)t}f(t)dt = F(s-a) \text{ }     $$

  • 第二移動定理 (t-shifting):     

    $$     \mathcal{L}[f(t-a)u_a(t)] = e^{-as}F(s) \text{ }     $$

        （計算過程）     $T=t-a$ と置換積分します。$t=a \to \infty$ のとき $T=0 \to \infty$ となります。    

    $$     \begin{aligned}     \mathcal{L}[f(t-a)u_a(t)] &= \int_a^\infty e^{-st}f(t-a)dt \\     &= \int_0^\infty e^{-s(T+a)}f(T)dT \text{ } \\     &= e^{-as} \int_0^\infty e^{-sT}f(T)dT = e^{-as}F(s) \text{ }     \end{aligned}     $$

       

4. sによる微分
#

$$\mathcal{L}[t^n f(t)] = (-1)^n \frac{d^n}{ds^n}F(s) \text{ }$$

（計算過程: n=1の場合） 積分のパラメータ微分を用います 。

$$ \begin{aligned} \frac{d}{ds}F(s) &= \frac{d}{ds}\int_0^\infty e^{-st}f(t)dt = \int_0^\infty \frac{\partial}{\partial s}(e^{-st}f(t))dt \text{ } \\ &= \int_0^\infty (-t e^{-st})f(t)dt \text{ } \\ &= -\int_0^\infty e^{-st}(tf(t))dt = -\mathcal{L}[tf(t)] \text{ } \end{aligned} $$

よって $\mathcal{L}[tf(t)] = -F’(s)$ となります 。

逆ラプラス変換
#

ラプラス変換された関数 $F(s)$ から、元の関数 $f(t)$ を見つける操作を逆ラプラス変換といい、 $\mathcal{L}^{-1}[F(s)]$ と表記します 。

  • レルヒ(Lerch)の定理: 2つの連続関数が同じラプラス変換を持つならば、それらの関数は完全に一致します 。これにより、逆ラプラス変換の一意性が保証されます。
  • 部分分数分解: 有理関数 $F(s)$ の逆ラプラス変換を求める際には、部分分数分解が有効な手法となります。     （計算例：$F(s) = \frac{s+7}{(s+1)(s+3)}$）     $F(s) = \frac{A}{s+1} + \frac{B}{s+3}$ とおきます。通分すると $s+7 = A(s+3) + B(s+1) = (A+B)s + (3A+B)$ 。     係数を比較して、$A+B=1, 3A+B=7$ を解くと $A=3, B=-2$ となります。     よって、$F(s) = \frac{3}{s+1} - \frac{2}{s+3}$ と分解できるので、逆ラプラス変換は     $$     f(t) = \mathcal{L}^{-1}\left[\frac{3}{s+1}\right] - \mathcal{L}^{-1}\left[\frac{2}{s+3}\right] = 3e^{-t} - 2e^{-3t} \text{ }     $$     となります。

常微分方程式への応用
#

ラプラス変換を用いると、定数係数の線形常微分方程式を、初期値を用いて簡単な代数方程式に変換して解くことができます。

解法の流れ 1.  微分方程式の両辺をラプラス変換する 。 2.  導関数の公式を用いて、$Y(s) = \mathcal{L}[y(t)]$ に関する代数方程式に変形する 。 3.  $Y(s)$ について解く 。 4.  $Y(s)$ を逆ラプラス変換して、解 $y(t)$ を求める 。

（計算例：$y’ + 2y = 0, \quad y(0)=3$） 1.  両辺をラプラス変換： $\mathcal{L}[y’ + 2y] = \mathcal{L}[0] \implies \mathcal{L}[y’] + 2\mathcal{L}[y] = 0$  2.  代数方程式へ： $(sY(s) - y(0)) + 2Y(s) = 0$  3.  $Y(s)$について解く： $(s+2)Y(s) = y(0) = 3 \implies Y(s) = \frac{3}{s+2}$  4.  逆ラプラス変換： $y(t) = \mathcal{L}^{-1}\left[\frac{3}{s+2}\right] = 3\mathcal{L}^{-1}\left[\frac{1}{s+2}\right] = 3e^{-2t}$ 

ラプラス変換の存在条件
#

関数 $f(t)$ のラプラス変換が存在するためには、いくつかの条件が必要です。

1.  区分的に連続 (Piecewise Continuous): 有限個の点を除いて連続であり、不連続点では有限な値を持つこと 。 2.  指数位数 (Exponential Order): ある定数 $M>0$ と $\gamma$ が存在し、十分に大きなすべての $t$ に対して $|f(t)| \le Me^{\gamma t}$ が成り立つこと 。

これらの条件を満たす関数は、 $s > \gamma$ の範囲で必ずラプラス変換が存在します 。

（存在証明の概略） $f(t)$ が上記の条件を満たすとき、ラプラス変換の積分が絶対収束することを示します 。

$$ \begin{aligned} \int_0^\infty |e^{-st}f(t)| dt &= \int_0^\infty e^{-st}|f(t)| dt \\ &\le \int_0^\infty e^{-st} (Me^{\gamma t}) dt \text{ } \\ &= M \int_0^\infty e^{-(s-\gamma)t} dt \text{ } \end{aligned} $$

$s > \gamma$ のとき、この積分は $M/(s-\gamma)$ に収束します 。絶対収束する積分はそれ自身も収束するため、ラプラス変換が存在することが示されます 。
