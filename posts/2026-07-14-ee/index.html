マクスウェル電磁気学
#

アンペールの法則と磁束保存則を用いて、磁気回路における「オームの法則」を導出する。また、媒質の不連続性（エアギャップ）を伴う磁気回路の境界値問題にふれる。詳しくは益川電磁気学でも読んでください。

1. 静磁場の方程式と電磁境界条件
#

定常電流および静止した磁性体が存在する系において、静磁場を支配するマクスウェル方程式は以下で与えられる。

$$\nabla \times \pmb{H} = \pmb{J}_{\text{f}}$$

$$\nabla \cdot \pmb{B} = 0$$

ここで、$\pmb{H}$ は磁界強度（$\text{A/m}$）、$\pmb{B}$ は磁束密度（$\text{T}$）、$\pmb{J}_{\text{f}}$ は自由電流密度（$\text{A/m}^2$）である。 媒質の構成方程式は以下の通りである。

$$\pmb{B} = \mu \pmb{H} = \mu_0 \mu_r \pmb{H}$$

透磁率 $\mu$ は媒質固有の定数であり、真空の透磁率 $\mu_0 = 4\pi \times 10^{-7} \ \text{H/m}$ と媒質の比透磁率 $\mu_r$ の積で表される。

異なる磁性体の境界（界面）における電磁境界条件は、上記の方程式から以下のように要請される。

  1. 磁束密度の法線成分の連続性: $$B_{1n} = B_{2n}$$
  2. 磁界強度の接線成分の連続性（境界上に自由面電流が存在しない場合）: $$H_{1t} = H_{2t}$$

2. 磁気回路近似
#

マクスウェルの方程式から巨視的な磁気回路モデルを導くにあたり、以下の近似を導入する。

2.1 漏れ磁束の無視
#

鉄心の比透磁率 $\mu_r$ が、周囲の空気（$\mu_{r,\text{air}} \approx 1$）に比べて十分に大きい（$\mu_r \gg 1$）と仮定する。 界面における磁束密度の法線成分の連続性 $B_{1n} = B_{2n}$ と構成方程式 $\pmb{B} = \mu \pmb{H}$ より、境界上で以下が成立する。

$$\mu_1 H_{1n} = \mu_2 H_{2n} \implies H_{2n} = \frac{\mu_1}{\mu_2} H_{1n}$$

鉄心（媒質1）から空気（媒質2）へ向かう磁界の法線成分について $H_{2n} = (\mu_1 / \mu_2) H_{1n}$ が成り立つ。$\mu_1 \gg \mu_2$ であるため、空気側では磁界はほぼ法線方向を向く。一方、鉄心内部では磁性体境界に沿う成分（接線成分）が支配的となる。 したがって、磁束は鉄心内部に閉じ込められ、外部への漏れ磁束は無視できる。このとき、磁路の任意の横断面 $A$ を貫く全磁束 $\Phi$ は磁路に沿って一定となる。

$$\Phi = \iint_{A} \pmb{B} \cdot d\pmb{S} = \text{const.} \quad [\text{Wb}]$$

2.2 一様磁界近似
#

磁路の断面の線度（代表的な太さ）が磁路の平均長 $l$ に比べて十分に小さい場合、鉄心内部の磁界 $\pmb{H}$ は各断面内で一様かつ磁路の接線方向を向くとみなせる。このとき、磁路に沿った周回積分は以下のように近似される。

$$\oint_{C} \pmb{H} \cdot d\pmb{l} \approx H l$$

3. 起磁力と磁束密度
#

平均磁路長 $l$、断面積 $A$、透磁率 $\mu$ の環状鉄心に、巻数 $N$ のコイルを巻いて電流 $I$ を流した系を考える。

3.1 起磁力 $F_m$
#

アンペールの法則の微分形に対し、磁路を周回する閉曲線 $C$ に沿って線積分を施し、ストークスの定理を適用する。

$$\oint_{C} \pmb{H} \cdot d\pmb{l} = \iint_{S} (\nabla \times \pmb{H}) \cdot d\pmb{S} = \iint_{S} \pmb{J}_{\text{f}} \cdot d\pmb{S} = NI$$

右辺は閉曲線 $C$ と交わる自由電流の総和であり、巻数 $N$ のコイルに電流 $I$ が流れている場合は $NI$ となる。この線積分量を起磁力 $F_m$ と定義する。

$$F_m = NI \quad [\text{A}]$$

3.2 磁界 $H$ と磁束密度 $B$
#

一様磁界近似のもとで、アンペールの積分則から鉄心内部の磁界の大きさ $H$ は以下のように求まる。

$$H = \frac{NI}{l} \quad [\text{A/m}]$$

この式から、磁界の強さ $H$ は磁路の単位長さあたりの起磁力に等しく、媒質の透磁率 $\mu$ に依存しない。 構成方程式を適用すると、鉄心内の磁束密度 $B$ は次式となる。

$$B = \mu H = \frac{\mu NI}{l} \quad [\text{T}]$$

全磁束 $\Phi$ は、磁束密度に断面積を乗じることで得られる。

$$\Phi = B A = \frac{\mu A N I}{l} \quad [\text{Wb}]$$

4. 透磁率と比透磁率
#

物質の磁気的性質は、比透磁率 $\mu_r$ に応じて以下のように分類される。

  1. 強磁性体（Ferromagnetic material）: $\mu_r \gg 1$ 外部磁界と同方向に強く磁化される物質。純鉄、ニッケル、フェライト、パーマロイなどが該当する。高い透磁率を持つため、磁気回路の鉄心材料として用いられる。
  2. 常磁性体（Paramagnetic material）: $\mu_r > 1$ （1よりわずかに大きい） 外部磁界と同方向に弱く磁化される物質。アルミニウムなどが該当する。
  3. 反磁性体（Diamagnetic material）: $\mu_r < 1$ （1よりわずかに小さい） 外部磁界と逆方向に弱く磁化される物質。銅、水、銀などが該当する。

4.1 代表的物質の比透磁率一覧
#

分類 物質名 比透磁率 $\mu_r$
反磁性 銅 (Cu) $0.99999756$
水 ($\text{H}_2\text{O}$) $0.99999004$
常磁性 空気 $1.000000365$
アルミニウム (Al) $1.000214$
強磁性 純鉄 (Fe) $200 \sim 8000$

パーマロイ (Permalloy) $8000 \sim 100000$

4.2 強磁性体における磁化の微視的機構
#

強磁性体が極めて高い比透磁率（$\mu_r \gg 1$）を示すのは、物質内部にスピンが平行に整列した「磁区（magnetic domain）」と呼ばれる微小な領域が存在するためである。外部磁界が印加されていない状態では、各磁区の磁化の向きはランダムであり、全体として磁化は打ち消し合っている。しかし、外部磁界 $\pmb{H}$ を印加すると、磁界の向きに沿った磁区の拡大（磁壁移動）や、磁区全体の向きの回転（磁化回転）が生じ、巨視的に非常に大きな磁束密度 $\pmb{B}$ が誘起される。これに対し、常磁性体や反磁性体では、熱振動によるスピンの乱れや軌道運動の反磁性応答のみが働くため、比透磁率はほぼ $1$ に留まる。

5. 磁気抵抗とオームの法則
#

式 (3.4) を起磁力 $F_m = NI$ を用いて書き直すと、

$$\Phi = \frac{NI}{\left(\dfrac{l}{\mu A}\right)}$$

分母の項を磁気抵抗 $R_m$ と定義する。

$$R_m = \frac{l}{\mu A} \quad [\text{H}^{-1} \text{ または } \text{A/Wb}]$$

これは、電気回路におけるオームの法則 $I = V / R$ と相似な、磁気回路における関係式を与える。

$$\Phi = \frac{F_m}{R_m}$$

6. エアギャップのある磁気回路
#

変圧器や回転機などの電磁機器では、磁路の一部に微小な間隙（エアギャップ）が設けられる。 鉄心部分の長さを $l_1$、透磁率を $\mu$ とし、エアギャップ部分の長さを $l_2$、透磁率を真空の透磁率 $\mu_0$ とする。

6.1 境界条件と磁抵抗の直列接続
#

ギャップ長 $l_2$ が十分に小さいと仮定すると、端部における磁束の広がり（端部効果）および漏洩磁束は無視できる。このとき、界面における磁束密度の法線成分連続条件（$B_{1n}=B_{2n}$）より、鉄心内部の磁束密度 $B_1$ とギャップ部の磁束密度 $B_2$ は等しくなる。

$$B_1 = B_2 = B = \frac{\Phi}{A}$$

磁路に沿ってアンペールの法則を適用すると、以下の関係が得られる。

$$H_1 l_1 + H_2 l_2 = NI$$

構成方程式を代入すると、

$$\frac{B}{\mu} l_1 + \frac{B}{\mu_0} l_2 = NI \implies \Phi \left( \frac{l_1}{\mu A} + \frac{l_2}{\mu_0 A} \right) = NI$$

鉄心部分の磁気抵抗を $R_{m1} = \frac{l_1}{\mu A}$、エアギャップ部分の磁気抵抗を $R_{m2} = \frac{l_2}{\mu_0 A}$ と定義すると、

$$\Phi (R_{m1} + R_{m2}) = NI$$

が得られる。これは電気回路における抵抗の直列接続に対応している。

6.2 磁位降下と回路方程式
#

アンペールの法則の周回積分 $\oint \pmb{H} \cdot d\pmb{l} = NI$ は、磁路をいくつかの区間に分割することで以下のように書き表せる。

$$\sum_{i} H_i l_i = NI$$

ここで、各区間における磁界強度と区間長の積 $U_i = H_i l_i$ は、電気回路における電圧降下に対応する量であり、これを磁位降下（Magnetic potential drop）と呼ぶ。 各区間の磁束密度 $B_i$ を用いて構成方程式 $\pmb{B} = \mu \pmb{H}$ および磁束 $\Phi = B_i A_i$ を適用すると、磁位降下は以下のように磁気抵抗 $R_{mi}$ と磁束 $\Phi$ の積として表される。

$$U_i = H_i l_i = \frac{B_i}{\mu_i} l_i = \left( \frac{l_i}{\mu_i A_i} \right) \Phi = R_{mi} \Phi$$

したがって、エアギャップを有する直列磁気回路全体の回路方程式は、

$$R_{m1} \Phi + R_{m2} \Phi = NI$$

となり、全体の起磁力 $NI$ が鉄心部分における磁位降下 $U_1 = R_{m1} \Phi = H_1 l_1$ と、エアギャップ部分における磁位降下 $U_2 = R_{m2} \Phi = H_2 l_2$ の和として分配されることを示している。

6.3 エアギャップが磁気回路に与える物理的影響
#

直列接続された鉄心部分の磁気抵抗 $R_{m1}$ とエアギャップ部分の磁気抵抗 $R_{m2}$ の比は、以下のように与えられる。

$$\frac{R_{m2}}{R_{m1}} = \frac{\mu_r l_2}{l_1}$$

この式から、エアギャップ長 $l_2$ が鉄心の長さ $l_1$ に比べて極めて小さい（$l_2 \ll l_1$）場合であっても、鉄心の比透磁率 $\mu_r$ が十分に大きければ、全体の磁気抵抗はエアギャップ部分によって支配されることがわかる。 例えば、$\mu_r = 1000$、鉄心長 $l_1 = 0.6 \ \text{m}$、ギャップ長 $l_2 = 2 \ \text{mm}$ の場合、抵抗比は、

$$\frac{R_{m2}}{R_{m1}} = \frac{1000 \times (2 \times 10^{-3})}{0.6} = \frac{10}{3} \approx 3.33$$

となり、わずか $2 \ \text{mm}$ の隙間が鉄心全体の $3$ 倍以上の磁気抵抗を持つ。このとき、起磁力 $NI$ の大半（全体の約 $77%$）はエアギャップ部分の磁化（磁束の維持）のために消費される。 工学的には、エアギャップを設けることで回路全体の合成磁気抵抗を意図的に引き上げ、過大な電流による鉄心の磁気飽和（磁束密度の頭打ち）を防止する設計が広く用いられている。

7. 演習
#

【例題 1】
#

以下のパラメータを持つ環状鉄心の磁気抵抗 $R_m$ を求めよ。

  • 鉄心の透磁率: $\mu = 5 \times 10^{-3} \ \text{H/m}$
  • 磁路の平均長さ: $l = 40 \ \text{cm} = 0.4 \ \text{m}$
  • 鉄心の断面積: $A = 6 \ \text{cm}^2 = 6 \times 10^{-4} \ \text{m}^2$

【解】: 磁気抵抗の定義式より、

$$R_m = \frac{0.4}{(5 \times 10^{-3}) \times (6 \times 10^{-4})} = \frac{0.4}{3 \times 10^{-6}} \approx 1.33 \times 10^5 \ \text{H}^{-1}$$

【例題 2】
#

以下のパラメータを持つ環状鉄心の透磁率 $\mu$ を求めよ。

  • 磁路の平均長さ: $l = 0.8 \ \text{m}$
  • 鉄心の断面積: $A = 30 \ \text{cm}^2 = 3 \times 10^{-3} \ \text{m}^2$
  • 磁気抵抗: $R_m = 3 \times 10^6 \ \text{H}^{-1}$

【解】: 磁気抵抗の定義式から透磁率 $\mu$ について解く。

$$\mu = \frac{l}{R_m A} = \frac{0.8}{(3 \times 10^6) \times (3 \times 10^{-3})} = \frac{0.8}{9 \times 10^3} \approx 8.89 \times 10^{-5} \ \text{H/m}$$

【例題 3】
#

比透磁率 $\mu_r = 1000$ のケイ素鋼の透磁率 $\mu$ を求めよ。ただし、真空の透磁率を $\mu_0 = 4\pi \times 10^{-7} \ \text{H/m}$ とする。

【解】: 定義式 $\mu = \mu_0 \mu_r$ より、

$$\mu = (4\pi \times 10^{-7}) \times 1000 = 4\pi \times 10^{-4} \approx 1.26 \times 10^{-3} \ \text{H/m}$$

【例題 4】
#

環状鉄心にコイルが巻かれた回路において、コイルに一定の電流を流したときの鉄心内の磁束は $\Phi_1 = 4 \times 10^{-4} \ \text{Wb}$ であった。この環状鉄心を取り除いて中空（空気）にし、同一の電流を流したところ、コイル内の磁束は $\Phi_2 = 2 \times 10^{-6} \ \text{Wb}$ に減少した。環状鉄心の比透磁率 $\mu_{r1}$ を求めよ。ただし、空気の比透磁率を $\mu_{r2} = 1$ とする。

【解】: 鉄心が存在する場合と、中空である場合の磁束はそれぞれ以下のように表される。

$$\Phi_1 = \frac{\mu_1 A N I}{l}, \quad \Phi_2 = \frac{\mu_2 A N I}{l}$$

幾何学的形状および巻線電流が同一であるため、磁束の比は透磁率の比に等しくなる。

$$\frac{\Phi_1}{\Phi_2} = \frac{\mu_1}{\mu_2} = \frac{\mu_{r1}}{\mu_{r2}}$$

したがって、鉄心の比透磁率 $\mu_{r1}$ は以下の通りとなる。

$$\mu_{r1} = \frac{\Phi_1}{\Phi_2} \mu_{r2} = \frac{4 \times 10^{-4}}{2 \times 10^{-6}} \times 1 = 200$$

【例題 5】
#

以下のパラメータを有する環状鉄心に電流を流したとき、鉄心内に生じる磁束 $\Phi$ と磁束密度 $B$ を求めよ。

  • 鉄心の透磁率: $\mu = 14.5 \times 10^{-4} \ \text{H/m}$
  • 鉄心の断面積: $A = 32 \ \text{cm}^2 = 3.2 \times 10^{-3} \ \text{m}^2$
  • 磁路の長さ: $l = 1.5 \ \text{m}$
  • コイル巻数: $N = 120$
  • コイル電流: $I = 12 \ \text{A}$

【解】: 全磁束 $\Phi$ は、次式より計算される。

$$\Phi = \frac{\mu A N I}{l} = \frac{(14.5 \times 10^{-4}) \times (3.2 \times 10^{-3}) \times 120 \times 12}{1.5} \approx 4.45 \times 10^{-4} \ \text{Wb}$$

また、磁束密度 $B$ は以下の通りとなる。

$$B = \frac{\Phi}{A} = \frac{4.45 \times 10^{-4}}{3.2 \times 10^{-3}} \approx 0.139 \ \text{T}$$

（あるいは $B = \frac{\mu N I}{l} = \frac{(14.5 \times 10^{-4}) \times 120 \times 12}{1.5} \approx 0.139 \ \text{T}$）

【例題 6】
#

平均磁路長 $l = 0.7 \ \text{m}$ の環状鉄心に、巻数 $N = 7000$ のコイルが巻かれている。コイルに電流 $I = 0.1 \ \text{A}$ を流したときの起磁力 $F_m$ と磁界の大きさ $H$ を求めよ。

【解】: 起磁力 $F_m$ の定義より、

$$F_m = N I = 7000 \times 0.1 = 700 \ \text{A}$$

磁界の大きさ $H$ は、アンペールの法則より、

$$H = \frac{NI}{l} = \frac{700}{0.7} = 1000 \ \text{A/m}$$

【例題 7】
#

図のような複合磁気回路について、以下の各値を求めよ。

  • コイル巻数: $N = 1200$
  • コイル電流: $I = 2.5 \ \text{A}$
  • 鉄心断面積: $A = 18 \ \text{cm}^2 = 1.8 \times 10^{-3} \ \text{m}^2$
  • 鉄心磁路長: $l_1 = 60 \ \text{cm} = 0.6 \ \text{m}$
  • 鉄心の比透磁率: $\mu_r = 1000$
  • エアギャップ長: $l_2 = 2 \ \text{mm} = 2 \times 10^{-3} \ \text{m}$
  • 真空の透磁率: $\mu_0 = 4\pi \times 10^{-7} \ \text{H/m}$

(1) 鉄心部分の磁気抵抗 $R_{m1}$ およびエアギャップ部分の磁気抵抗 $R_{m2}$ (2) 磁路を貫く全磁束 $\Phi$ (3) 磁束密度 $B$ (4) 鉄心内部の磁界の強さ $H_1$

【解】:

(1) 各部分の磁気抵抗 鉄心部分の磁気抵抗 $R_{m1}$ は、

$$R_{m1} = \frac{l_1}{\mu_0 \mu_r A} = \frac{0.6}{(4\pi \times 10^{-7}) \times 1000 \times (1.8 \times 10^{-3})} = \frac{0.6}{7.2\pi \times 10^{-7}} = \frac{10^7}{12\pi} \approx 2.65 \times 10^5 \ \text{H}^{-1}$$

エアギャップ部分の磁気抵抗 $R_{m2}$ は、

$$R_{m2} = \frac{l_2}{\mu_0 A} = \frac{2 \times 10^{-3}}{(4\pi \times 10^{-7}) \times (1.8 \times 10^{-3})} = \frac{2 \times 10^6}{7.2\pi} = \frac{10^7}{36\pi} \approx 8.84 \times 10^5 \ \text{H}^{-1}$$

(2) 回路全体の磁束 $\Phi$ 全体の合成磁気抵抗 $R_m$ は、

$$R_m = R_{m1} + R_{m2} \approx 2.653 \times 10^5 + 8.842 \times 10^5 = 1.149 \times 10^6 \ \text{H}^{-1}$$

起磁力は $F_m = N I = 1200 \times 2.5 = 3000 \ \text{A}$ であるため、磁束 $\Phi$ は、

$$\Phi = \frac{F_m}{R_m} = \frac{3000}{1.149 \times 10^6} \approx 2.61 \times 10^{-3} \ \text{Wb}$$

(3) 磁束密度 $B$

$$B = \frac{\Phi}{A} = \frac{2.61 \times 10^{-3}}{1.8 \times 10^{-3}} \approx 1.45 \ \text{T}$$

(4) 鉄心内部の磁界の強さ $H_1$

$$H_1 = \frac{B}{\mu_0 \mu_r} = \frac{1.45}{(4\pi \times 10^{-7}) \times 1000} \approx 1.15 \times 10^3 \ \text{A/m}$$

補足：磁束密度の直接導出
#

上記の計算において、磁性体界面の条件から磁束密度 $B$ を直接決定する式を導出しておくことで、個々の磁気抵抗を介さずに値を算出できる。 鉄心とエアギャップで磁束密度 $B$ が等しいことから、アンペールの法則より次式が成り立つ。

$$B \left( \frac{l_1}{\mu_0 \mu_r} + \frac{l_2}{\mu_0} \right) = NI$$

これを $B$ について解くと、次式が得られる。

$$B = \frac{\mu_0 \mu_r NI}{l_1 + \mu_r l_2}$$

ここで、分母の $l_{\text{eff}} = l_1 + \mu_r l_2$ は、エアギャップ長を鉄心の比透磁率を用いて換算した実効的な磁路長に対応する。 数値を代入すると、

$$l_{\text{eff}} = 0.6 + 1000 \times (2 \times 10^{-3}) = 2.6 \ \text{m}$$

を得る。これより、磁束密度 $B$ は以下のように直接計算できる。

$$B = \frac{(4\pi \times 10^{-7}) \times 1000 \times 3000}{2.6} = \frac{1.2\pi}{2.6} \approx 1.45 \ \text{T}$$

磁束 $\Phi$ は $\Phi = B A$、鉄心内部の磁界 $H_1$ は $H_1 = B / (\mu_0 \mu_r) = NI / l_{\text{eff}}$ よりただちに求まる。この手法は、個別に磁気抵抗を計算する際の中間値の丸め誤差を防ぐとともに、計算過程の簡素化に有効である。

8. 電気回路との対応
#

支配方程式および構成方程式の形式的類似性から、電気回路と磁気回路には以下のような対応関係が成り立つ。

項目 電気回路 磁気回路
ポテンシャル 起電力 $V = \int \pmb{E} \cdot d\pmb{l} \ [\text{V}]$ 起磁力 $F_m = \oint \pmb{H} \cdot d\pmb{l} = NI \ [\text{A}]$
流束密度 電流密度 $\pmb{J} \ [\text{A/m}^2]$ 磁束密度 $\pmb{B} \ [\text{T}]$
総流束 電流 $I = \iint \pmb{J} \cdot d\pmb{S} \ [\text{A}]$ 磁束 $\Phi = \iint \pmb{B} \cdot d\pmb{S} \ [\text{Wb}]$
媒質の輸送係数 導電率 $\sigma \ [\text{S/m}]$ 透磁率 $\mu \ [\text{H/m}]$
回路抵抗 電気抵抗 $R = \frac{l}{\sigma A} \ [\Omega]$ 磁気抵抗 $R_m = \frac{l}{\mu A} \ [\text{H}^{-1}]$
オームの法則 $I = \frac{V}{R}$ $\Phi = \frac{F_m}{R_m}$
連続の式 $\sum I = 0 \quad (\nabla \cdot \pmb{J} = 0)$ $\sum \Phi = 0 \quad (\nabla \cdot \pmb{B} = 0)$
周回積分則 $\sum V = \sum R I \quad (\nabla \times \pmb{E} = \pmb{0})$ $\sum F_m = \sum R_m \Phi \quad (\nabla \times \pmb{H} = \pmb{J}_{\text{f}})$

8.1 アナロジーを利用した公式導出の単純化
#

磁気回路の問題を解くにあたり、磁気回路固有の公式を個別に暗記する必要はない。以下の対応ルールを適用することで、電気回路の公式からすべての磁気回路の関係式を再現できる。

  1. 基本パラメータの置換: 電気回路における起電力 $V$、電流 $I$、電流密度 $J$、導電率 $\sigma$ を、それぞれ磁気回路における起磁力 $F_m = NI$、磁束 $\Phi$、磁束密度 $B$、透磁率 $\mu$ に置き換える。
  2. 抵抗の公式: 電気抵抗の式 $R = \frac{l}{\sigma A}$ において $\sigma \rightarrow \mu$ と置換することで、磁気抵抗の式 $R_m = \frac{l}{\mu A}$ が得られる。
  3. オームの法則: 電気回路における $I = \frac{V}{R}$ に上記の置換を施すことで、磁気回路のオームの法則 $\Phi = \frac{F_m}{R_m}$ が得られる。
  4. 電界と磁界の対応: 電界の強さが平均的に $E = \frac{V}{l}$ で与えられるのと同様に、磁界の強さは $H = \frac{F_m}{l} = \frac{NI}{l}$ となる。この関係式は透磁率 $\mu$ を含まないため、一様磁路における磁界の強さは起磁力と磁路長のみから決定できる。

このように、磁気回路の計算は電気回路におけるオームの法則および合成抵抗の知識（直列接続における $R = R_1 + R_2$ など）をそのまま翻訳する操作に帰着されるため、暗記すべき数式を大幅に削減できる。
