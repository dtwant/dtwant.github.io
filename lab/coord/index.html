COORD_TRANSFORM

COORD_TRANSFORM // TENSOR_CALCULUS
#

入力された座標値やベクトル場成分を、Nerdamer（数式処理システム）を用いて別の座標系へリアルタイムかつ厳密に変換する動力学計算機です。

  • [ POINT TRANSFORM ] x=1, y=1, z=0 を極座標へ変換するなどの点変換。
  • [ VECTOR TRANSFORM ] ベクトル場 $A(x,y) = -y \hat{i} + x \hat{j}$ を極座標ベクトル $A(r,\theta)$ の成分へ代数的（シンボリック）に変換・簡約化。

[!NOTE] AVAILABLE SYSTEMS:

  • [ CARTESIAN ] デカルト座標 $(x, y, z)$
  • [ POLAR ] 極座標 (2D) $(r, \theta)$
  • [ CYLINDRICAL ] 円筒座標 $(r, \theta, z)$
  • [ SPHERICAL ] 球座標 $(r, \theta, \phi)$

（旧バージョンの各系における微積分公式や計量テンソルは、画面下部の REFERENCE_DB から確認できます。）
