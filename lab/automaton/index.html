AUTOMATON_VIZ

AUTOMATON_VIZ // REGEX_ENGINE
#

正規表現を入力するとNFA/DFAの状態遷移図を自動生成し、文字列マッチングをステップ実行で可視化するツール。

[!NOTE] AUTOMATON MODULES:

  • [ REGEX_PARSE ] 正規表現→構文木→NFA（Thompson構成法）
  • [ NFA_VIEW ] 非決定性有限オートマトンの状態遷移図
  • [ DFA_CONVERT ] 部分集合構成法によるDFA変換
  • [ STEP_EXEC ] 入力文字列のステップ実行マッチング
  • [ TRANSITION_TABLE ] 状態遷移表のテーブル表示

プリセット ((a|b)abb / [0-9]+ / ab+c? / (ab|cd)*ef)
