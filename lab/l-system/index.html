L-System Fractal Plant

22 March 2026·132 words·1 min·
Author
dt
bio

L-SYSTEM GEN

Fractal Plant Generator v2

SYSTEM_HELP // L-SYSTEM v2

L-System（リンデンマイヤー・システム）は、文字列の置換ルールを使って植物やフラクタル図形の成長をシミュレートするアルゴリズムです。

🌿 はじめに

  1. 上部の プリセットボタン（Organic / Classic / Dragon / Crystal / Koch / Sierp.）からお好みの形を選択
  2. GENERATE ボタンで描画します

🎛️ パラメータの調整

  • Angle: 枝分かれの角度（小さいと細かい、大きいと広がる）
  • Iterations: 反復回数（多いほど複雑になるが重くなる）
  • Length: 枝の長さ
  • Organic Mode: ONにすると、自然な揺らぎと葉が付きます

🌱 成長アニメーション

  • 🌱 GROW ボタンを押すと、iterationを1から段階的に増やして成長する様子をアニメーションで表示します

🔍 ズーム & 移動

  • ズーム: マウスホイール / 右下の+−ボタン / スマホでピンチ
  • 移動: ドラッグ / スワイプ
  • RST: ズームをリセット

🎨 カラーテーマ

  • ドロップダウンから 4種類のテーマ（Cyber Blue→Green / Fire / Sakura Pink / Terminal Mono）を切り替え

💾 保存

  • SAVE PNG: 現在の描画をPNG画像としてダウンロード

📱 スマホでは左下の [ SYSTEM_MENU ] でパネルを開きます。
