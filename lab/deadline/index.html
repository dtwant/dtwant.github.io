Deadline Manager

提出期限と推定所要時間を入力し，締め切りまでの時間を管理する.

_DEADLINE_SURVIVAL
SYSTEM_TIME: --:--

SYSTEM_GUIDE
#

  1. TASK_NAME: 課題名を入力
  2. DATE / PERIOD: 提出日と時限（1〜4限）を選択
  3. EST (MIN): 完了にかかる推定時間を「分」で入力
  4. ADD: タスク登録

STATUS_CODE
#

  • GREEN: 安全圏.
  • YELLOW: 猶予 24時間未満.
  • ORANGE: 猶予 3時間未満.
  • RED (BLINKING): CRITICAL WARNING. 残り時間が作業時間を下回っているため，着手推奨.

> LOCAL_STORAGE_CONNECTED. DATA_PERSISTENCE_ACTIVE.
