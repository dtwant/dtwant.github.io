Fractal Explorer

22 March 2026·100 words·1 min·
Author
dt
bio

DEEP_ZOOM_OS

Fractal Explorer // JS-64bit
MANDELBROT
MAGNIFICATION 1.0 x
CENTER -0.500, 0.000
BOOKMARKS
COLOR_MATRIX
ITERATIONS (200)

SYSTEM_HELP // FRACTAL v2

マンデルブロ集合とジュリア集合を探索するフラクタルビューアです。無限の深さまでズームして、美しいフラクタル模様を発見しましょう。

🖱️ PC での操作

  1. ズームイン: マウスホイールを上にスクロール、またはダブルクリック
  2. ズームアウト: マウスホイールを下にスクロール
  3. 移動: ドラッグ（クリックしたまま引っ張る）

📱 スマホでの操作

  1. ズーム: 2本指でピンチイン/ピンチアウト
  2. 移動: 1本指でスワイプ

🎨 カラーパレット

  • 下部パネルのドロップダウンから 5種類のカラー（Cyber Neon / Matrix / Inferno / Mono / Deep Ocean）を切り替え
  • ITERATIONS スライダーで計算精度を調整（高いほど精細だが遅くなる）

🔮 ジュリア集合モード

  • ピンクの JULIA SET ボタンで切り替え。現在のマンデルブロ座標に基づくジュリア集合を描画します
  • もう一度押すとマンデルブロに戻ります

📌 ブックマーク & 保存

  • 左パネルの + BOOKMARK で現在の座標・ズームを保存。クリックで呼び出し
  • SAVE PNG で現在の画面をPNG画像としてダウンロード
  • RESET で初期位置に戻ります

📱 スマホではHUDパネルを閉じるには ☰ ボタンや CTRL ボタンを使います。
