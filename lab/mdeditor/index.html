Markdown Editor

NEURAL_EDITOR // ADVANCED_TEXT_PROCESSING_UNIT
#

サイバーパンク環境に特化した、マークダウン対応の最強テキストエディタ。 ブラウザ上に３つの独立した記憶スロット（SLOT 1〜3）を持ち、リロードしてもデータは自動復元されます。

[!TIP] OVERKILL_FEATURES (特殊機能):

  • ⚡ ZEN_MODE: 執筆中の行以外を暗く落とし、極限の集中状態を作ります
  • ✨ SPARKS: 打鍵に連動してサイバー粒子が飛び散るエフェクト（処理が重い場合はOFF推奨）
  • ⏱ HISTORY: 数分おきに自動でスナップショットを取得。右側のパネルから過去の状態へタイムトラベル可能。
  • 📐 MERMAID_DIAGRAM: ` ` `mermaid で囲んでテキストを書くと、自動でフローチャートやシーケンス図に変換されます。
