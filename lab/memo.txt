📐 物理ソルバー系ツール 提案
1. 🧮 PHYSICS_SOLVER — 高校物理ステップ解法エンジン
条件（質量、速度、角度、摩擦係数…）を入力すると、使うべき公式を自動選択し、式変形の過程を一つずつ表示して最終解に至る。

例	動作イメージ
斜面上の物体の運動	→ 力の分解 → 運動方程式 ma=mgsinθ−μmgcosθ → 加速度 a=... → 到達時間 t=...
エネルギー保存	→ 条件判定「非保存力なし」→ (1/2)mv² = mgh → v=√(2gh)
円運動	→ 向心力の式 → 張力/垂直抗力との関係 → 速度条件の導出
機能: 分野選択（力学/波動/電磁気/熱/原子）→ 問題タイプ選択 → パラメータ入力 → LaTeX整形されたステップバイステップ解説出力

2. ⚙️ LAGRANGIAN_SOLVER — ラグランジアン解法ツール
高校レベルの力学問題を、あえてラグランジュ形式で解く全過程を表示するツール。「ニュートン力学でも解けるけど、解析力学ではこう解く」を見せる。

ステップ	表示内容
① 一般化座標の選択	「自由度1 → 一般化座標 θ を選ぶ」
② 座標の表現	x = l sinθ, y = -l cosθ
③ 運動エネルギー T	T = (1/2)ml²θ̇² （導出過程付き）
④ ポテンシャル U	U = -mgl cosθ
⑤ ラグランジアン L	L = T - U = ...
⑥ オイラー・ラグランジュ方程式	d/dt(∂L/∂θ̇) - ∂L/∂θ = 0 → 展開
⑦ 運動方程式	ml²θ̈ + mgl sinθ = 0
⑧ 近似・解の解釈	sinθ ≈ θ → 単振動 ω = √(g/l)
プリセット: 単振り子 / 斜面上の物体 / ばね振り子 / 二重振り子 / アトウッドの機械 / ビーズがワイヤ上を滑る問題 etc.

3. 📏 DIMENSION_CHECKER — 次元解析・単位整合性チェッカー
物理の式を入力すると、各項の次元を分解し、等式の次元が合っているかを検証するツール。さらに、次元解析からπ定理を使って公式を導出する機能も。

機能	説明
式の次元検証	F = mv²/r → [MLT⁻²] = [M][LT⁻¹]²[L⁻¹] = [MLT⁻²] ✓
次元不一致の検出	E = mv → [ML²T⁻²] ≠ [MLT⁻¹] ✗ どこが間違いか指摘
バッキンガムπ定理	物理量のリストから無次元量を構成、公式の形を推定
各物理定数の次元表	G, ℏ, k_B, ε₀ 等の次元を一覧表示
使い所: レポートや試験前に自分の式の整合性を素早くチェック

4. 🔄 COORD_TRANSFORM — 座標変換・テンソル計算ツール
デカルト ↔ 極座標 ↔ 円筒座標 ↔ 球座標の変換を、変換過程を式展開付きで表示。各座標系でのgrad, div, rot, ラプラシアンの表式も導出・表示。

機能	説明
座標変換	(x,y,z) → (r,θ,φ) の変換式と逆変換を明示
ヤコビアン導出	座標変換のヤコビ行列・ヤコビアンを計算過程付きで表示
ベクトル演算子の導出	∇f, ∇·F, ∇×F を選んだ座標系で展開
計量テンソル	各座標系の g_ij を表示、線素 ds² を導出
具体例の計算	例: 球座標でのラプラス方程式の分離変数法の概略
5. 📊 ERROR_ANALYSIS — 誤差伝播計算ツール
実験レポート作成に必須。測定値と誤差を入力し、誤差伝播公式を自動導出、最終結果の誤差を計算。

機能	説明
誤差伝播の自動計算	f(a,b,c) の式と各変数の誤差 → δf を偏微分から導出表示
有効数字の処理	入力値に応じた有効数字の丸め方を解説
最小二乗法フィッティング	データ点を入力 → 直線/曲線フィット、χ²値、残差プロット
統計量の計算	平均、標準偏差、標準誤差、信頼区間を一括計算
レポート用出力	v = (3.42 ± 0.15) m/s 形式のLaTeX出力
6. ⚡ CIRCUIT_SOLVER — 回路方程式ソルバー
回路のトポロジーを入力すると、キルヒホッフの法則から連立方程式を立て、解く過程を表示するツール。

ステップ	表示内容
① 回路図の認識	ノード・ループの識別
② KCL（電流則）	各ノードでの電流保存式を立式
③ KVL（電圧則）	各ループでの電圧降下式を立式
④ 連立方程式	行列形式で表示
⑤ 解	各枝の電流・各点の電位を導出
対応: 直流抵抗回路 / RC・RL・RLC過渡応答 / 交流インピーダンス計算

まとめ：方向性の比較
ツール	コンセプト	レベル

LAGRANGIAN_SOLVER	同じ問題を解析力学で解く	高校→学部橋渡し
DIMENSION_CHECKER	式の次元整合性チェック	高校〜学部
COORD_TRANSFORM	座標変換・微分演算子の導出	学部
ERROR_ANALYSIS	実験誤差の伝播計算	高校〜学部実験
CIRCUIT_SOLVER	キルヒホッフ→連立方程式→解法	高校〜学部
